# 🛒 Flipkart Selenium Automation Project

## 📌 Project Overview

This project is an end-to-end automation framework built using **Java + Selenium WebDriver** to automate basic user flows on Flipkart.

It demonstrates core automation concepts like:

* Browser automation
* Element interaction
* Window handling
* Wait strategies
* Test structuring

---

## 🚀 Tech Stack

* **Language:** Java
* **Automation Tool:** Selenium WebDriver
* **Build Tool:** Maven
* **IDE:** Eclipse

## ⚙️ Setup Instructions

### 1️⃣ Clone the Repository

```
git clone https://github.com/your-username/flipkart-selenium-automation.git
cd flipkart-selenium-automation
```

### 2️⃣ Install Dependencies

```
mvn clean install
```

### 3️⃣ Run the Project

* Open in Eclipse
* Run any class (e.g., `AddToCart.java`) as **Java Application**
